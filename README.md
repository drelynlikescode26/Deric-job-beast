# Deric-job-beast

This repository contains the Auto-Applier project for autonomous job application automation.

The core implementation lives in the `Auto-Applier` directory.

## Quick start

```bash
cd /Users/deric/Deric-job-beast/Auto-Applier
pip install -r requirements.txt
playwright install
cp .env.example .env
# fill in environment variables
python main.py --scout-now
```
